# 🛡️ CYBER TECH - Dark Neon Website

**Teknolojia ya Kisasa, Apps, Tools, na Cybersecurity Tips**

## 📱 Overview

Cyber Tech ni website ya kisasa yenye muonekano wa dark neon (nyeusi + green glow) inayofanana na app ya simu. Inahusiana na teknolojia, apps, tools na cybersecurity tips.

## ✨ Features

### 🎨 Design
- **Dark Neon Theme** - Muonekano wa nyeusi na green glow effects
- **Responsive Design** - Inafanya kazi vizuri kwenye simu, tablet, na desktop
- **Glow Cards** - Cards zenye hover effects na neon glow
- **Smooth Animations** - Transitions za kuona vizuri

### 📑 Sections (5 Main Pages)

1. **Home** 🏠
   - Welcome banner
   - Featured cards na quick links
   - Overview ya website

2. **News** 📰
   - Habari za teknolojia
   - Cybersecurity tips
   - Tech trends

3. **Apps** 📱
   - Recommendations za apps mbalimbali
   - App ratings na descriptions
   - Download links

4. **Tools** 🛠️
   - Online tools zenye kumfaida
   - Image compressor
   - JSON formatter
   - Password generator
   - Na zaidi...

5. **Profile** 👤
   - User profile taarifa
   - Statistics
   - About section
   - Contact options

### 🧭 Navigation
- **Header** - Logo, search bar
- **Bottom Navigation** - 5 icons kwa sections (kama mobile app)
- **Keyboard Shortcuts**:
  - `Alt+H` - Home
  - `Alt+N` - News
  - `Alt+A` - Apps
  - `Alt+T` - Tools
  - `Alt+P` - Profile

## 🚀 How to Use

### 1. Local Development
```bash

git clone https://github.com/wewem0065-svg/cyber-tech.git

# Open in browser
open index.html
# OR
start index.html
```

### 2. Deploy on GitHub Pages

1. Go to repository settings
2. Scroll to "GitHub Pages"
3. Select "main" branch
4. Save
5. Access at: `https://wewem0065-svg.github.io/cyber-tech/`
